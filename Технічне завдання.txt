Назва: OpenClub
Кодова назва: OC
Терміни розробки: 07.02.2022-07.05.2032
Назва організації розробника: Організація
Назва організації замовника: Замовник

Загальні відомості:

	Закритий блог по запрошенням з можливістю викладати пости. При реєстрації на блог потрібно вказати запрошення, щоб створити свій аккаунт. У кожного нового користувач буде 2 запрошення, які він 	може розповсюдити. Користувач має можливість створювати, коментувати пости та підписуватися на інших
 	користувачів.


Технічний план: 

	Сторінка входу:
		Поля для вводу нікнейму, паролю.
		Кнопка “ввійти”
		Кнопка “створити новий акаунт”
	Сторінка реєстрації:
		Поля для вводу нікнейму, паролю, запрошення.
		Кнопка “зареєструвати”
	Сторінка редагування:
		Поля для зміни нікнейму, паролю.
		Кнопка для зміни фотографії профілю.
		Кнопка видалити профіль.
		Кнопка для переходу на домашню сторінку.
	Головна сторінка:
		Поле пошуку, кнопка для переходу на домашню сторінку, мініатюрна аватарка.
		Пости людей на яких підписаний користувач.
		Пост: мініатюрна аватарка, нік є кнопкою для переходу на домашню сторінку автора поста, кнопка для написання коментаря, кнопка для перегляду коментарів
	Домашня сторінка:
		Кнопка-посилання на головну сторінку, профіль сторінку, пошук.
		Кнопка для створення нового поста. 
		Список минулих постів.
		Кнопка для редагування постів.



Характеристики користувачів:
	Звичайний користувач:
		Створювати, коментувати пости, підписатись та відписатись від користувача.
	Адмін:
		Права звичайного користувача, видаляти будь-які пости, видаляти або банити користувачів.
